# **App Name**: StreamAssist

## Core Features:

- User Input: Accept user prompts through a text input field.
- AI-Powered Response Generation: Generate responses using a pre-trained language model based on the user's input.
- Contextual Memory (tool): Maintain a conversation history to provide contextual relevance to the AI's responses.
- Response Display: Display the AI-generated responses in a chat-like interface.
- Persistent state using Streamlit: Store information in order to remember states across reruns in Streamlit.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to convey intelligence and sophistication.
- Background color: Light gray (#EEEEEE) to provide a clean and neutral backdrop.
- Accent color: Electric indigo (#6F38C5) to highlight interactive elements and call-to-actions.
- Body and headline font: 'Inter', a sans-serif, for a clean, modern, and readable interface.
- Use minimalist, line-based icons to represent different features and actions.
- Employ a clean, single-column layout for easy readability and user flow.
- Incorporate subtle animations for transitions and loading states to improve the user experience.
